Netcentric Final Integration Project

This website was made as a final project for Netcentic Computing at Seattle Pacific University. The following describe the functions of the various options in the navigation bar above.

Display All: Displays all the book objects stored in the MongoDB database using a Node.js API as the backend.

Find Book: Allows you to search for a specific book using its ISBN or the author's name.

Add New Book: Provides the functionality to add any book into the MongoDB database.

Remove Book: Gives the ability to remove any book from the database using its isbn.

Update Book: Allows you to update any of the available books in the database.
